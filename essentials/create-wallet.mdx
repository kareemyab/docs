---
title: '/create-wallet'
description: 'Creates a new Solana wallet using Crossmint service and links it to a user ID for easy management.'
icon: 'wallet'
openapi: 'POST /create-wallet'
---

<Info>
**Rate Limit:** 100 requests per 15 minutes
</Info>

<Warning>
**⚠️ ONE-TIME OPERATION PER USER**: Each user should only create **one** Crossmint wallet. Once created, use the same `walletAddress` for all future file registrations. Creating multiple wallets per user is unnecessary and not recommended.
</Warning>

---

## What This Endpoint Does

<Steps>
  <Step title="Creates Crossmint Wallet">
    Generates a new Solana wallet using Crossmint's API-key signing method
  </Step>
  <Step title="Stores User Relation">
    Creates an on-chain record linking your `userID` to the wallet address
  </Step>
  <Step title="Returns Details">
    Provides the wallet address and transaction confirmation
  </Step>
</Steps>

---

## Parameters

<ResponseField name="userID" type="string" required>
  Unique identifier for the user
  
  **Constraints:** Maximum 50 characters
  
  Example: `"user_12345"`
</ResponseField>





## Response Examples

### Success Response (201)

<ResponseField name="message" type="string">
  Success confirmation message
  
  Example: `"Success! Your Crossmint wallet has been created and linked to your user ID."`
</ResponseField>

<ResponseField name="details" type="object">
  Wallet creation details
  
  <Expandable title="Details Object">
    <ResponseField name="walletAddress" type="string">
      The newly created Solana wallet address
      
      Example: `"9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM"`
    </ResponseField>
    
    <ResponseField name="chain" type="string">
      Blockchain network identifier
      
      Example: `"solana"`
    </ResponseField>
    
    <ResponseField name="transactionSignature" type="string">
      On-chain transaction signature for the wallet creation
      
      Example: `"2nBhEBYYvfaAe16UMNqRHre4YNSskvuYgx3M6E4JP1oDYvZEJHvoPzyUwkoB6VRYuVEMarHNr9RxJWnyb8FnFnBg"`
    </ResponseField>
    
    <ResponseField name="explorerUrl" type="string">
      Direct link to view the transaction on Solana Explorer
      
      Example: `"https://explorer.solana.com/tx/...?cluster=devnet"`
    </ResponseField>
  </Expandable>
</ResponseField>

### Common Error Responses

<CodeGroup>
```json Missing User ID (400)
{
  "message": "Hold on! We need a `userID` to proceed."
}
```

```json Server Error (500)
{

  "details": {

{

}
```
</CodeGroup>
