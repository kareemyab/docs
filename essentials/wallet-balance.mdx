**Rate Limit:** 100 requests per 15 minutes  
**Content-Type:** `application/json`
</Info>

---

## What This Endpoint Does

<Steps>
  <Step title="Address Validation">
    Validates the provided wallet address format using Solana's Base58 encoding
  </Step>
  <Step title="Balance Query">
    Queries the Solana network for the wallet's current SOL balance
  </Step>
  <Step title="Format Conversion">
    Converts the balance from lamports to SOL for human-readable display
  </Step>
  <Step title="Network Information">
    Includes cluster information (devnet, mainnet, etc.) in the response
  </Step>
</Steps>

---

## Parameters

<ResponseField name="walletAddress" type="string" required>
  Valid Solana wallet address to check balance for
  
  **Constraints:** Must be a valid Solana public key (Base58 format)
  
  Example: `"9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM"`
</ResponseField>

## Understanding SOL and Lamports

### Currency Conversion

<Info>
**1 SOL = 1,000,000,000 lamports**

- **SOL**: Human-readable decimal format (e.g., 1.5 SOL)
- **Lamports**: Smallest unit, always integers (e.g., 1,500,000,000 lamports)
</Info>

### Balance Interpretation

<Tabs>
  <Tab title="New Wallets">
    **Balance**: 0 SOL (0 lamports)
    
    **Meaning**: Wallet exists but has no funds. Cannot pay transaction fees.
    
    ```json
    {
      "balanceSOL": 0,
      "balanceLamports": 0
    }
    ```
  </Tab>
  
  <Tab title="Funded Wallets">
    **Balance**: > 0 SOL
    
    **Meaning**: Wallet has funds and can perform transactions.
    
    ```json
    {
      "balanceSOL": 2.5,
      "balanceLamports": 2500000000
    }
    ```
  </Tab>
  
  <Tab title="Low Balance Warning">
    **Balance**: < 0.01 SOL
    
    **Meaning**: Wallet has minimal funds, may not cover transaction fees.
    
    ```json
    {
      "balanceSOL": 0.005,
      "balanceLamports": 5000000
    }
    ```
  </Tab>
</Tabs>

## Integration Examples

### Balance Verification Before Transactions

```typescript
async function checkSufficientBalance(walletAddress: string, minimumSOL: number = 0.01) {
  const response = await fetch('/wallet-balance', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ walletAddress })
  });
  
  const data = await response.json();
  
  if (data.details.balanceSOL < minimumSOL) {
    throw new Error(`Insufficient balance: ${data.details.balanceSOL} SOL (need at least ${minimumSOL} SOL)`);
  }
  
  return data.details.balanceSOL;
}
```

### Multi-Wallet Monitoring

```typescript
async function checkMultipleWallets(wallets: string[]) {
  const balances = await Promise.all(
    wallets.map(async (wallet) => {
      const response = await fetch('/wallet-balance', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ walletAddress: wallet })
      });
      const data = await response.json();
      return {
        wallet,
        balance: data.details.balanceSOL,
        status: data.details.balanceSOL > 0.01 ? 'healthy' : 'low'
      };
    })
  );
  
  return balances;
}
```
