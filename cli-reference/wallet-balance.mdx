# wallet-balance

Get the SOL balance of a specified wallet address.

## Usage

```bash
core-api-cli wallet-balance [options]
```

## Required Options

<ParamField path="--walletAddress, -w" type="string" required>
  Wallet address to check balance for
</ParamField>

## Optional Options

<ParamField path="--json" type="boolean">
  Output the result as JSON instead of formatted text
</ParamField>

## Examples

### Basic Balance Check

```bash
core-api-cli wallet-balance \
  --walletAddress "9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM"
```

### JSON Output for Automation

```bash
core-api-cli wallet-balance \
  --walletAddress "9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM" \
  --json
```

## Error Handling

Common error scenarios:

- **Invalid wallet address**: Ensure the wallet address is a valid Solana public key
- **Network issues**: Check your internet connection and API availability
- **Non-existent wallet**: The command will return a balance of 0 for wallets that don't exist

## Use Cases

### Quick Balance Check

```bash
# Check balance before performing operations
core-api-cli wallet-balance --walletAddress "9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM"
```

### Automation Scripts

```bash
#!/bin/bash
WALLET_ADDRESS="9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM"
balance=$(core-api-cli wallet-balance --walletAddress "$WALLET_ADDRESS" --json | jq -r '.details.balanceSOL')

if (( $(echo "$balance < 0.01" | bc -l) )); then
  echo "Warning: Low balance ($balance SOL)"
else
  echo "Balance sufficient: $balance SOL"
fi
```

### Multi-Wallet Monitoring

```bash
#!/bin/bash
wallets=("wallet1..." "wallet2..." "wallet3...")

for wallet in "${wallets[@]}"; do
  echo "Checking $wallet..."
  core-api-cli wallet-balance --walletAddress "$wallet"
  echo ""
done
```

## Notes

- Balance is returned in both SOL (decimal) and lamports (integer) formats
- 1 SOL = 1,000,000,000 lamports
- The command works with any valid Solana wallet address
- Network cluster information is included in the response (devnet, mainnet, etc.)
