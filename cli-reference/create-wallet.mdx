# create-wallet

Create a new wallet for a user using the Trust Engine Core API.

## Usage

```bash
core-api-cli create-wallet [options]
```

## Required Options

<ParamField path="--userID, -u" type="string" required>
  User ID for the new wallet
</ParamField>

## Optional Options

<ParamField path="--json" type="boolean">
  Output the result as JSON instead of formatted text
</ParamField>

## Examples

### Basic Wallet Creation

```bash
core-api-cli create-wallet --userID "user_12345"
```

### JSON Output for Automation

```bash
core-api-cli create-wallet --userID "user_12345" --json
```

## Error Handling

Common error scenarios:

- **Missing User ID**: Ensure you provide a valid user ID
- **User ID already exists**: The user may already have a wallet associated
- **Network issues**: Check your internet connection and API availability

## Notes

- Each user ID can only have one wallet associated with it
- The created wallet is a Crossmint wallet on the Solana blockchain
- The wallet is automatically linked to the provided user ID
- Save the wallet address for future operations
