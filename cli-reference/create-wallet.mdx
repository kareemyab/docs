# Trust Engine create-wallet

> Create a new wallet for a user

The create-wallet command allows you to create a new Crossmint wallet for a user in the Trust Engine system. Each user should only run this command **once** and reuse the generated wallet for all future operations.

To use it, run the following command from your terminal

```bash
trust-engine-cli create-wallet [options]
```

## Options

| Option               | Description                                      |
| -------------------- | ------------------------------------------------ |
| `-u, --userID <id>`  | User ID for the new wallet (alphanumeric string) |
| `--json`             | Output the result as JSON                        |

## Example

Create a new wallet for a user:

```bash
trust-engine-cli create-wallet --userID "user_12345"
```
