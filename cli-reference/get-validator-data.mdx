# Trust Engine get-validator-data

> Get a validator's data from the Trust Engine system

The get-validator-data command allows you to retrieve detailed information about a Trust Engine validator, including their reputation score, voting history, and performance metrics within the Trust Engine validation network.

To use it, run the following command from your terminal

```bash
trust-engine-cli get-validator-data [options]
```

## Options

| Option                      | Description                                       |
| --------------------------- | ------------------------------------------------- |
| `-w, --walletAddress <address>` | Wallet address of the validator (Solana public key) |
| `--json`                    | Output the result as JSON                         |

## Example

Get validator data for a specific wallet address:

```bash
trust-engine-cli get-validator-data --walletAddress "9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM"
```
