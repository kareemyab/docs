# CLI Reference

The Trust Engine CLI is a command-line tool that provides direct access to all Core API functionality. It's perfect for automation, scripting, and quick operations without writing code.

## Installation

Install the CLI globally using npm:

```bash
npm install -g @trust-engine/cli
```

## Getting Started

Once installed, you can use the CLI with the `core-api-cli` command:

```bash
core-api-cli --help
```

## Global Options

All commands support these global options:

- `--json` - Output results in JSON format for programmatic use
- `--help` - Show help information for any command

## Examples

### Quick File Registration
```bash
core-api-cli register \
  --file ./document.pdf \
  --contentTitle "Important Document" \
  --walletAddress "9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM" \
  --walletType "crossmint"
```

### JSON Output for Scripting
```bash
core-api-cli create-wallet \
  --userID "9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM" \
  --json
```

## Configuration

The CLI connects to `http://localhost:3005` by default. This assumes you have a local development server running. For production usage, you may need to modify the source code to point to your production API endpoint.

## Error Handling

All commands return appropriate exit codes:
- `0` - Success
- `1` - Error (check the error message for details)

This makes the CLI perfect for use in shell scripts and automation pipelines.
